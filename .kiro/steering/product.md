# Product Overview

LinkNest is a modern, full-stack social media platform that enables users to connect, share content, and communicate in real-time.

## Core Features

- User authentication and profile management
- Post creation, editing, and deletion with image uploads
- Real-time messaging between users via WebSocket
- Social interactions: follow/unfollow, likes, comments
- Real-time notifications for user activities
- Friend request system with pending/accepted states
- Dark/Light mode theming
- Responsive design for mobile and desktop

## User Experience

- Clean, modern UI built with Chakra UI component library
- Real-time updates without page refreshes
- Image uploads handled via Cloudinary CDN
- Persistent authentication using JWT tokens
- Optimized for performance with code splitting and lazy loading

## Deployment

- Production deployment on Render platform
- MongoDB Atlas for database hosting
- Cloudinary for image storage and CDN
- Environment-based configuration for dev/staging/production
